Everything You Need

همه چیزی که برای ساخت ربات‌های قدرتمند سروش پلاس نیاز دارید

⚡

Fully Asynchronous

Built from the ground up with Python's asyncio. High performance, non-blocking I/O for concurrent operations.

🔒

No API Key Required

Uses built-in Soroush Plus credentials. No need to register for API keys — just install and start building.

🎯

Event-Driven Handlers

Powerful event system with decorators. Handle messages, edits, callbacks, inline queries, and more.

🧩

Composable Filters

Chain filters with & (AND), | (OR), ~ (NOT). Create complex conditions with clean, readable code.

🏷

Inline & Reply Buttons

Build interactive keyboards with inline and reply buttons. Support for URL buttons, callback data, and more.

💬

Conversation API

Interactive conversation flows with context managers. Perfect for multi-step bot dialogs and user input collection.

🔄

Finite State Machine

Built-in FSM for managing conversation state. Multiple storage backends, decorator-based routing.

🛠

Plugin System

Dynamic plugin loading from directories. Modular bot architecture that scales with your project.

🧠

Middleware Support

Pre/post processing of updates with middleware. Add logging, rate limiting, and custom logic transparently.

⏰

Built-in Scheduler

Schedule messages and callbacks with interval or one-shot timing. Never miss a reminder again.

💾

Multiple Storage Backends

Memory, SQLite, Redis, PostgreSQL — choose the backend that fits your deployment needs.

🛡

Rate Limiting

Token bucket algorithm with automatic flood wait handling. Stay within API limits without manual intervention.

🏘

AFK Auto-Reply

Smart auto-responder with per-chat rate limiting, cooldowns, and configurable max replies per session.

🛣

Chat Administration

High-level admin API: ban, mute, pin, purge, bulk actions, admin logs — all in clean async methods.

🔀

Message Mirroring

Real-time message cloning between chats. Strip forward headers, add prefixes, filter content.

📥

File Transfer with Progress

Upload and download files with progress tracking. Chunked transfers for large files.

📝

Sync Wrapper

Use without async/await via spluspy.sync. Same API, zero async complexity for simple scripts.

🌐

Batch Operations

Send, delete, forward multiple messages at once. Efficient batched API calls with configurable delays.

📊

Professional Logging

Structured, namespaced loggers with color support. Log events, requests, errors, security, and performance.

Start in Seconds

Get a working Soroush Plus bot in just a few lines of code.

در چند خط کد، یک ربات سروش پلاس کارآمد بسازید

bot.py 
from spluspy import Client, filters

bot = Client("my_session")

@bot.on_message(filters.command("start"))
async def start_handler(client, message):
    await message.reply("Hello! I'm your bot.")

@bot.on_message(filters.private & filters.text)
async def echo_handler(client, message):
    await message.reply(
        f"You said: {message.text}"
    )

bot.run()
filters_demo.py 
from spluspy import Client, filters

bot = Client("my_session")

@bot.on_message(filters.command("help"))
async def help_cmd(client, msg):
    await msg.reply("Available commands:")

@bot.on_message(filters.photo)
async def photo_handler(client, msg):
    await msg.reply("Nice photo!")

@bot.on_message(filters.regex(r"(\d+)"))
async def number_handler(client, msg):
    num = msg.pattern_match.group(1)
    await msg.reply(f"Found: {num}")

bot.run()

Installation

Install SplusPy with the features you need.

SplusPy را با قابلیت‌های مورد نیازتان نصب کنید

Basic

pip install spluspy

Core library with all essential features.

Fast Encryption

pip install spluspy[speed]

Performance-optimized with cryptg & orjson.

Redis Backend

pip install spluspy[redis]

Add Redis storage for shared state.

PostgreSQL

pip install spluspy[postgres]

Add PostgreSQL storage for enterprise use.

All Dependencies

pip install spluspy[all]

Everything including dev tools & testing.

Sync Usage

from spluspy.sync import Client

No async/await needed — just import and use.
